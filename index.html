Technical Details

Spotify API with Spring-Boot as back-end

The server is based on Java's Spring-Boot and hosted on Heroku, it serves the client by providing a link to Spotify's music API and returning the data following a REST approach.

The API returns information about artists and their albums, although due to a limitation on Spotify's public API not allowing a search based on genre, the original entries (names) were hardcoded onto the server.

HTML, CSS 3 and JQuery as front-end

Front-end built with HTML and CSS, using CSS's Flex-box for position, with transitions and effects.

Content comes from the Heroku app and uses the Fetch API to retrieve albums and artists, and, while the first has some limitations, the app can retrieve a large number of artists. It is also possible to search for any artist or album, even if not related to main theme of the app.
